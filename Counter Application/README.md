## рџ’Ў How It Works

- The counter starts with an initial value of **0**.
- Clicking **Increment** increases the counter by **1**.
- Clicking **Decrement** decreases the counter by **-1**, 
- Clicking **Reset** restores the counter to its initial value (**0**).
